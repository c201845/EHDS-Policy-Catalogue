# EHDS Secondary Use Policy Catalogue

This repository provides a machine-readable **ODRL + DPV Policy Catalogue** designed for the European Health Data Space (EHDS). 
It operationalizes the legal requirements for secondary health data use through a modular framework.

## 📂 Repository Structure

- **/permissive**: Contains policies defining authorized secondary use purposes and permitted data categories.
- **/prohibitive**: Contains policies for prohibited uses and individual data subject opt-outs.
